# bootAgent
Local setup for running a AI agent, capable of reading files, writing files and running Python programs within a bound directory. 
Agent runs on Google Gemini AI model. A personal API-key is required to run the agent (See https://aistudio.google.com/app/ or one of the links there).
This setup imports the key from a .env file located in the root of this directory. For security reasons, this .env file has not been shared here, as it contains my personal key.
